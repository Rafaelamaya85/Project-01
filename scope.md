# Scope Document

## Project Purpose

The purpose of this project is to create an introductury webpage for CMPA 3301 that introduces me as a student, my educational goals, background, professional ambitions and interests while demonstrating efficient use of foundational HTML skills.

## Description

This project will consist of a single webpage developed using basic HTML. The webpage will introduce me as a student in the Computing Applications program at Texas Tech University and provide details about my academic journey, future career aspirations, and hobbies. The page will be organized using semantic HTML elements such as headings, paragraphs, lists, and hyperlinks to create a clear and professional structure.

## Desired Results

- Develop a webpage that clearly communicates who I am as a student and professional.
- Demonstrate the ability to organize information using appropriate HTML elements.
- Create a webpage that is easy to read and understand for users.
- Provide brief details about my academic accomplishments, future educational plans, career objectives as well as highlight personal interests.
- Successfully publish the project to GitHub and verify that all files are accessible.
- Gain valuable experience creating and editing HTML documents.

## Exclusions

- The project will not feature complex styling or interactive content. 
- The project is not intended to build on my profesional portfolio but instead serve as valuable experience.
- The project will not contain multiple pages or navigation tabs of any kind.
- The project will not focus on visual design.

The main focus of this project is to understand and demonstrate core HTML concepts to create a simple and professional introduction webpage.
